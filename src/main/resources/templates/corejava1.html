What is Core Java?

Code Java is the technologies and application programming interfaces (APIs) that make up the Java Platform, Standard Edition(Java SE)

There were five primary goals that Java was created to meet

1. It had to be simple, object-oriented, and familiar

2. It had to be "robust and secure"

3. It must be "architecture-neutral and portable

4. It must execute with "high performance"

5. It must be "interpreted, threaded, and dynamic"

In addition it promised to be (WORA) Write Once, Run Anywhere
