What is Git

Git is a distributed version control tool used to store code, track updates, and share among programmers

What is GitHub

GitHub is a cloud based platform, currently owned by Microsoft, that is one of the vehicles that can store Git code

Git is open source and completely independent of Github and runs locally on your computer

GitHub can be free or is available for a fee and competes with other cloud based services

Git is primarily used from the command line, although there is a gui interface available

There are also other version control systems to choose from such as Subversion
